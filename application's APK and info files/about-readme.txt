The application scratches/crawls the website using Jsoup and fetches the text and images present in it. 
HTTP requests being processed with the help of Volley.
Webpage crawled with the help of JSoup.
Images being displayed with the help of Picasso. 
The textBox and the Images being displayed are in different frames and will be scrolled separately.

The following Link was used for primary testing: https://www.autocar.co.uk/car-review/porsche/718-cayman-gts
